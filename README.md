# WEB-ENGINEERING-5th-SEMESTER-Project-
**NoorApp** is a browser-based Islamic companion app built with HTML, CSS, and JavaScript, offering Quran reading, Prayer Times, Duas, Tasbeeh Counter, Zakat Calculator, Qibla Finder, Hadith, and Islamic Calendar — all in one place.
